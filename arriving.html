Arriving Checklist

Storage / Garage Code
24671 + ENTER  (ENTER again to close)
  • Enter storage area: code 24671 then ENTER. Press ENTER again to close.
  • Retrieve the front door key and elevator key from the key rack on the wall inside the storage area (to the right).
  • The outside elevator door is called by turning the key in the lock and releasing.
  • Check that the main water valve is ON — blue handle next to the water heater. Handle parallel to pipe = ON; perpendicular = OFF.
  • Check that the water heater temperature panel is set to "A".
  • Set thermostats on each floor using the touchscreen. Tap a temperature and swipe up/down to adjust. Use the ceiling fans too.
  • Turn on the under-counter ice machines in the kitchen and cabana bar via the switch panel inside each unit (if not already running). First batch: ~20–30 min. Full fill: ~12 hours.
